WeatherPy Overview

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, I pulled weather data from the OpenWeatherMap API on December 5, 2017 to assemble a dataset on 500 randomly selected cities.

After assembling the dataset, I used Matplotlib to plot latitude against various aspects of the weather: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends.
